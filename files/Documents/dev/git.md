
# Git commands

## Create/switch to OR move uncommitted changes to new branch
git switch -c <new-branch>

## Discard unstaged changes
git restore .

## Exit commit message without changes being applied
:cq!

## Clone repo without history, i.e. only the top of the tree
git clone --depth 1 git@github.com:...




# Misc

## View file history on GitHub
https://github.com/jvalcher/my_app/commits/<branch>/<file>
