# Neovim

```
j6 - navigate by relative numbers
b  - jump to start of previous word
w  - jump to start of next word
e  - jump to end of word
E  - jump to end of block of non-space characters
va" - select between, including token
```

<leader>r - copy relative file path
<leader>p - copy absolute file path

<localleader>g - cwd string search
<localleader>f - cwd file search


## Install version from source
```
git pull
git fetch
git checkout v0.10.4
sudo make distclean
make
sudo make install
```
